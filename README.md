# ToDo list bot

Telegram-bot to make and use(?) ToDo lists
Written on **Python 3.13** by using **pyTelegramBotAPI** and **SQLite3**.

## Possibilities
- Create ToDo list with name
- Delete/Show/Add commands
- Using SQLite tables and interact with them
